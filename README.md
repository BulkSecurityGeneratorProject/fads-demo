# fads-demo
Demonstration and Testing project for the TWD team to refine Test Integtration, Code Branching,  and Code Customization of Openstack Java applications

The fads-demo is a jhipster generated application the include the following domain objects.

    Organization
    Person
    Location
    Roleorganizationperson- Many-to-Many relationship between Organization and Person
    Organizationlocation - Many-to-Many relationship between Organization and Location
    Personlocation - Many-to-Many relationship between Organization and Person
